📊 Google Search Trend Analysis

ChatGPT vs Gemini vs Claude (India – Last 12 Months)

Project Overview ;

This project analyzes Google search trends of three major AI tools — ChatGPT, Gemini, and Claude — using the Google Trends API. The objective is to compare search interest, identify patterns, and generate insights based on user search behavior in India.

🛠 Tools & Technologies

Python

Google Colab

Pandas

Matplotlib

Seaborn

PyTrends (Google Trends API)


📌 Methodology

Extracted 12 months of search trend data using PyTrends

Cleaned and structured the dataset

Performed statistical analysis (mean, max, growth trends)

Created visualizations (line charts & correlation heatmap)

Conducted region-wise trend analysis


📈 Key Insights

ChatGPT dominates overall search interest in India.

Gemini shows noticeable spikes during product announcements.

Claude maintains steady but comparatively lower search volume.

Strong positive correlation between ChatGPT and Gemini trends.
